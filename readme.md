# ☀️ SunCart – Summer Essentials Store


## 📌 Project Purpose

SunCart is a modern summer-themed eCommerce website where users can explore and purchase seasonal products like sunglasses, skincare, beach accessories, outfits, footwear, and more.

The project focuses on beautiful UI design, authentication system, protected routes, responsive layout, and smooth user experience.

---

## 🚀 Key Features

### 🔝 Navbar
- Logo branding
- Home, Products, My Profile links
- Login / Register buttons
- User avatar after login
- Logout system

### 🌅 Home Page
- Animated hero slider
- Summer Sale Banner
- Popular products section
- Summer care tips
- Top brands section

### 🛍️ Products Page
- Show all summer products
- Category filter buttons
- Dynamic product cards

### 🔒 Protected Product Details
- Must login first
- Redirect back after login

### 🔐 Authentication
- Register new user
- Login existing user
- Google login (demo)
- Error handling with toast

### 👤 My Profile
- View user photo
- Name and email
- Update profile information

### 📱 Fully Responsive
- Mobile friendly
- Tablet friendly
- Desktop optimized
